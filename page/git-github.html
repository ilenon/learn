Git & Github

git init

Inicia o o git no projeto.

git add (git add * || git add file.ext)

Adiciona o conteúdo do projeto ao container.

git status

Verifica se há alterações e o que está no container.

git commit (git commit -m "comments")

Prepara o conteúdo para ser enviado ao repositório remoto.

git log

Traz um log de todo o processo do projeto.

rm -rf .git

Remove um repositório git iniciado.
